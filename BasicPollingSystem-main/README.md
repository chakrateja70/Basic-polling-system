# BasicPollingSystem
Create dotenv file in server:

PORT = 3000 <br/>
JWT_SECRET = jwtSecretKey <br/>
DB_HOST= "localhost" <br/>
DB_USER= "root" <br/>
DB_PASSWORD= <password> <br/>

</ul>
